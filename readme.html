Используемые программы

Этот архив содержит исходники прошивки, что бы их открыть, вам потребуется среда разработки Code Vision AVR.

В архие есть скомпилированная пошивка spec_keyb.hex.

Этот архив содержит схемы и печатные платы. Что бы их открыть, вам потребуется программа Proteus.

Печатная плата так же сохранена в формате Microsoft Word.

Введение

В двух словах. Прошиваете микроконтроллер. Выбираете скорсоть 8 Мгц. Подпаиваете к микроконтроллеру провода от компьютера, 4 провода от PS/2 клавиатуры. Включаете компьютер и получаете удовольствие.

Собственная раскладка

Программа "Настройка раскладки" позволяет создать свою привязку клавиш. Программа формирует массив чисел, который необходимо вставить в исходники, скомпилировать исходники и прошить микроконтроллер.

Настройка раскладки во время работы

Эта прошивка может изменять привязку клавиш РС к Специалисту во время работы. Для этого надо нажать на нужную кнопку (или повернуть ручку джойстика) и нажать BREAK. Светодиоды должны последовательно мигнуть. Затем набираете цифрами код клавиши РК86 и нажимаете ENTER.

Цифры я напишу позже. Цифры можно подсмотреть в программе "Настройка раскладки".

Цифра 0 отвязывает кнопку.

Цифра 65534 заполняет энергонезависимую память значениями по умолчанию.

Цифра 65535 сохраняет изменения в энергонезависимую память. При этом, сохраняет выбраную раскладку.
